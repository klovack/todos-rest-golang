# Todo List Rest API

Simple todo list rest API using golang. It just create a HTTP server without the database.

## Endpoints
- ### GET /api/todos
    Get All todos from the mockup
- ### GET /api/todos/{id}
    Get todo by id
- ### POST /api/todos
    Create new todo
- ### PUT /api/todos/{id}
    Update todo by id
- ### DELETE /api/todos/{id}
    Delete todo by id
